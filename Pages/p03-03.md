# Heteroscedasticity

## ğŸŒŸ **1. What is Heteroscedasticity? (Layman Explanation)**

âœ… **Imagine you are driving on a road.**

* **Part 1:** Smooth and calm â†’ No bumps (low variance).
* **Part 2:** Suddenly lots of bumps â†’ Rough (high variance).

ğŸ”‘ **Heteroscedasticity means:**
The **variability (variance) changes over time**.

---

### ğŸ§  **In Time Series:**

If **data fluctuation (volatility) increases or decreases over time**, it is **heteroscedastic**.

âœ”ï¸ Common in **financial time series (stock prices)** where market volatility changes over time.

---

### âœ… **Example**

* Stock market during **normal period**: low volatility.
* Stock market during **crisis or news events**: high volatility.

---

## ğŸ”¬ **2. Technical Definition**

* **Homoscedasticity:** Constant variance over time.
* **Heteroscedasticity:** Variance changes over time.

**ARCH effect (Autoregressive Conditional Heteroscedasticity):**

* When variance today **depends on past squared errors (shocks)**, indicating volatility clustering.

---

### ğŸ’¡ **Why is it Important?**

âœ… Most basic models assume **constant variance (homoscedasticity)**.
âœ… For **better forecasting**, especially in finance, we model **changing volatility** using **ARCH or GARCH models**.

---

## ğŸ§ª **3. How to Test for Heteroscedasticity?**

### **Common Test:**

ğŸ”¬ **ARCH Test (Lagrange Multiplier Test)**

* Null Hypothesis (H0): **No ARCH effect** (variance is constant).
* If **p-value < 0.05**, reject H0 â†’ **Heteroscedasticity is present**.

---

## ğŸ **4. How to Perform ARCH Test using Python**

Here is a **step-by-step practical example**:

---

### ğŸ”· **Step 1. Install and import required libraries**

```python
# Install via: pip install arch

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
from arch import arch_model
from statsmodels.stats.diagnostic import het_arch
```

---

### ğŸ”· **Step 2. Create sample data**

```python
# Simulated data with volatility clustering
np.random.seed(42)
returns = np.random.normal(0, 1, 100)
for i in range(1, 100):
    if i > 50:
        returns[i] += np.random.normal(0, 3)  # Increase variance after time 50

# Convert to pandas Series
returns = pd.Series(returns)

# Plot returns
returns.plot(title='Simulated Returns with Changing Volatility')
plt.show()
```

---

### ğŸ”· **Step 3. Perform ARCH Test**

```python
# Perform ARCH test (Lagrange Multiplier test for heteroscedasticity)
test_result = het_arch(returns)

# Unpack result
lm_statistic, lm_pvalue, f_statistic, f_pvalue = test_result

print("LM Statistic:", lm_statistic)
print("LM p-value:", lm_pvalue)

# Interpretation
if lm_pvalue < 0.05:
    print("Conclusion: Heteroscedasticity is present (ARCH effect detected).")
else:
    print("Conclusion: No significant heteroscedasticity detected.")
```

---

### ğŸ”· **Step 4. Modeling with GARCH if heteroscedasticity is present**

If **ARCH effect is detected**, model with **ARCH/GARCH** for better volatility forecasting:

```python
# Fit GARCH(1,1) model
model = arch_model(returns, vol='Garch', p=1, q=1)
garch_fit = model.fit(disp='off')

# Print summary
print(garch_fit.summary())
```

---

âœ”ï¸ **How to check?** Use **ARCH test** in Python.

# GARCH Modeling

## ğŸŒŸ **1. What is GARCH? (Layman Explanation)**

âœ… **Imagine stock market prices.**

* Some days are calm â†’ low volatility (small ups and downs).
* Some days are turbulent â†’ high volatility (big jumps up or down).

ğŸ”‘ **GARCH models this changing volatility** to:

* Predict **future risk (variance)**.
* Useful for **financial forecasting, risk management, option pricing**.

---

### ğŸ§  **What does GARCH stand for?**

* **G**eneralized
* **A**utoregressive
* **C**onditional
* **H**eteroscedasticity

âœ”ï¸ **ARCH model:** Variance depends on **past squared errors**.
âœ”ï¸ **GARCH model:** Variance depends on **past squared errors + past variances** (more general and accurate).

---

## ğŸ”¬ **2. Technical Definition**

The **GARCH(1,1)** model:

* **Current variance** depends on:

  * **Constant term (Ï‰)**
  * **Yesterdayâ€™s squared error (Î±1)**
  * **Yesterdayâ€™s variance (Î²1)**

---

### âœ… **Why GARCH is Important?**

âœ”ï¸ Most financial time series have **volatility clustering** (high volatility follows high, low follows low).
âœ”ï¸ **GARCH captures this pattern** and predicts future volatility.

---

## ğŸ **3. GARCH Modeling in Python (Step-by-Step)**

### ### ğŸ”· **Step 1. Install and import libraries**

```python
# Install via: pip install arch

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
from arch import arch_model
```

---

### ğŸ”· **Step 2. Load or simulate financial return data**

Here we will **simulate returns with volatility clustering** for practice.

```python
np.random.seed(42)
returns = np.random.normal(0, 1, 1000)

# Introduce volatility clustering after index 500
for i in range(500, 1000):
    returns[i] += np.random.normal(0, 3)

# Convert to pandas Series
returns = pd.Series(returns)

# Plot returns
returns.plot(title='Simulated Returns with Volatility Clustering', figsize=(10,4))
plt.show()
```

---

### ğŸ”· **Step 3. Check for ARCH effect (heteroscedasticity)**

```python
from statsmodels.stats.diagnostic import het_arch

test_result = het_arch(returns)
print("ARCH Test p-value:", test_result[1])

if test_result[1] < 0.05:
    print("Conclusion: Heteroscedasticity detected. Proceed with GARCH modeling.")
else:
    print("No significant heteroscedasticity detected.")
```

---

### ğŸ”· **Step 4. Fit GARCH(1,1) model**

```python
# Define GARCH(1,1) model
model = arch_model(returns, vol='Garch', p=1, q=1)

# Fit the model
garch_fit = model.fit()

# Print summary
print(garch_fit.summary())
```

---

### ğŸ”· **Step 5. Interpret Output**

Key outputs:

* **Ï‰ (omega):** Constant variance term.
* **Î±1 (alpha1):** Effect of yesterdayâ€™s shock.
* **Î²1 (beta1):** Effect of yesterdayâ€™s variance.

âœ… **If Î±1 + Î²1 is close to 1:** High persistence â†’ shocks affect volatility for long time.

---

### ğŸ”· **Step 6. Forecasting Volatility**

```python
# Forecast future volatility
forecast = garch_fit.forecast(horizon=5)
print("Forecasted Variance:\n", forecast.variance[-1:])
```

---

### ğŸ”· **Step 7. Plot Conditional Volatility**

```python
# Plot conditional volatility (predicted volatility)
garch_fit.plot(annualize='D')
plt.show()
```
