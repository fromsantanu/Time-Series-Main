### ğŸ§  **Why Check Residual Normality?**

âœ”ï¸ **If residuals are normal:**

## ğŸ”¬ **2. Technical Definition**

âœ… **Structural Normality Test (Residual Normality Test)** checks:

* Whether the **residuals of your model follow a normal distribution**.

---

### **Common Tests:**

1. **Jarque-Bera Test**

   * Tests for normality using **skewness and kurtosis**.
   * Null Hypothesis (H0): Residuals are normally distributed.

2. **Shapiro-Wilk Test**

   * Tests for normality (more powerful for small samples).
   * H0: Residuals are normally distributed.

3. **Q-Q Plot (Quantile-Quantile plot)**

   * Visual check for normality.

---

## ğŸ **3. How to Perform Normality Tests on Residuals using Python**

Here is a **step-by-step practical example**:

---

### ğŸ”· **Step 1. Install and import libraries**

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import statsmodels.api as sm
from scipy.stats import shapiro, jarque_bera
```

---

### ğŸ”· **Step 2. Create data and fit a simple time series model**

Here we create data and fit an **ARIMA model** as an example.

```python
# Simulate data with trend and noise
np.random.seed(42)
x = np.arange(100)
y = 2 + 0.5*x + np.random.normal(0,5,100)

# Convert to pandas Series
data = pd.Series(y)

# Fit ARIMA model
model = sm.tsa.ARIMA(data, order=(1,1,1)).fit()

# Get residuals
residuals = model.resid
```

---

### ğŸ”· **Step 3. Perform Shapiro-Wilk Test**

```python
stat, p_value = shapiro(residuals)

print("Shapiro-Wilk Test Statistic:", stat)
print("p-value:", p_value)

# Interpretation
if p_value < 0.05:
    print("Conclusion: Residuals are NOT normally distributed.")
else:
    print("Conclusion: Residuals are normally distributed.")
```

---

### ğŸ”· **Step 4. Perform Jarque-Bera Test**

```python
jb_stat, jb_pvalue = jarque_bera(residuals)

print("Jarque-Bera Test Statistic:", jb_stat)
print("p-value:", jb_pvalue)

# Interpretation
if jb_pvalue < 0.05:
    print("Conclusion: Residuals are NOT normally distributed.")
else:
    print("Conclusion: Residuals are normally distributed.")
```

---

### ğŸ”· **Step 5. Visual Normality Check using Q-Q Plot**

```python
sm.qqplot(residuals, line='s')
plt.title("Q-Q Plot of Residuals")
plt.show()
```

âœ… **Interpretation (Layman):**

* If points lie along the straight line â†’ **residuals are normal**.
* If points deviate significantly â†’ **residuals are not normal**.
