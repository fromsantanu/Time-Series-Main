# Normality Test

## ðŸŒŸ **1. What is Normality Test? (Layman Explanation)**

âœ… **Imagine you are shooting arrows at a target.**

* If your arrows are **scattered evenly around the center**, your errors are normal.
* If arrows cluster away from center (e.g. top-right corner), errors are **not normal**.

ðŸ”‘ **In time series modeling:**

* **Residuals = errors = difference between predicted and actual values.**

âœ”ï¸ **Normality tests check** if these residuals are **normally distributed (bell-shaped curve)**.

---

### ðŸ§  **Why is Normality Important?**

* Many statistical models (e.g. **ARIMA**) assume residuals are **normally distributed** with constant mean (0) and variance.
* **Non-normal residuals â†’ model may be mis-specified** â†’ poor forecasting.

---

### âœ… **Example**

After fitting ARIMA:

1. **Check residuals plot â†’ random scatter.**
2. **Check residuals histogram â†’ bell curve.**
3. **Check normality test â†’ p-value > 0.05 â†’ residuals are normal.**

---

## ðŸ”¬ **2. Technical Definition**

### **Normality Tests:**

1. **Shapiro-Wilk Test**

   * H0: Data is **normal**.
   * If p-value < 0.05 â†’ Reject H0 â†’ Data is **not normal**.

2. **Kolmogorov-Smirnov (KS) Test**

3. **Jarque-Bera Test**

   * Based on **skewness and kurtosis** of data.

---

## ðŸ **3. How to Perform Normality Tests on Residuals in Python**

Here is a **step-by-step practical example**:

---

### ðŸ”· **Step 1. Import libraries and create sample time series model**

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
from statsmodels.tsa.arima.model import ARIMA
from scipy.stats import shapiro, kstest, normaltest, jarque_bera

# Simulated time series data
np.random.seed(42)
data = np.random.normal(0, 1, 100)
df = pd.Series(data)
```

---

### ðŸ”· **Step 2. Fit ARIMA model**

```python
# Fit ARIMA model
model = ARIMA(df, order=(1,0,0))
model_fit = model.fit()

# Get residuals
residuals = model_fit.resid

# Plot residuals
plt.figure(figsize=(10,4))
plt.subplot(1,2,1)
plt.plot(residuals)
plt.title("Residuals Time Series Plot")

plt.subplot(1,2,2)
plt.hist(residuals, bins=20)
plt.title("Residuals Histogram")
plt.show()
```

---

### ðŸ”· **Step 3. Perform Normality Tests**

#### âœ… **(a) Shapiro-Wilk Test**

```python
stat, p = shapiro(residuals)
print("Shapiro-Wilk Test p-value:", p)

if p > 0.05:
    print("Residuals are normally distributed (Fail to reject H0).")
else:
    print("Residuals are NOT normally distributed (Reject H0).")
```

---

#### âœ… **(b) Jarque-Bera Test**

```python
jb_stat, jb_p, skew, kurtosis = jarque_bera(residuals)
print("Jarque-Bera Test p-value:", jb_p)

if jb_p > 0.05:
    print("Residuals are normally distributed (Fail to reject H0).")
else:
    print("Residuals are NOT normally distributed (Reject H0).")
```

---

#### âœ… **(c) Dâ€™Agostinoâ€™s K-squared Test (Normaltest)**

```python
stat, p = normaltest(residuals)
print("Dâ€™Agostinoâ€™s K-squared Test p-value:", p)

if p > 0.05:
    print("Residuals are normally distributed (Fail to reject H0).")
else:
    print("Residuals are NOT normally distributed (Reject H0).")
```

---

### ðŸ”· **Step 4. Interpretation**

* **p-value > 0.05 â†’ residuals are normal â†’ model is adequate.**
* **p-value < 0.05 â†’ residuals not normal â†’ consider model improvement or transformation.**
